# Análisis de Evasión de Clientes (Churn)
## Descripción del Proyecto

Este proyecto tiene como objetivo analizar el fenómeno de evasión de clientes (Churn) en una empresa de servicios de telecomunicaciones.

A través de técnicas de limpieza, transformación y análisis exploratorio de datos (EDA), se identificaron patrones asociados a la cancelación del servicio, con el fin de generar recomendaciones estratégicas orientadas a la retención de clientes.

## Objetivos

1. Comprender la distribución de la variable Churn.

2. Identificar variables numéricas y categóricas asociadas a la cancelación.

3. Detectar perfiles de clientes con mayor riesgo de evasión.

4. Proponer acciones estratégicas basadas en evidencia.

## Tecnologías Utilizadas

- Python

- Pandas

- NumPy

- Matplotlib

- Jupyter Notebook

## Estructura del Proyecto

El análisis se desarrolló en un notebook estructurado en las siguientes etapas:

- Importación de datos

- Limpieza y tratamiento

- Conversión de variables

- Renombramiento de columnas

- Creación de variables derivadas

- Análisis exploratorio de datos (EDA)

- Distribución general del churn

- Análisis de variables numéricas

- Análisis de variables categóricas

- Visualizaciones gráficas

- Conclusiones e insights

- Recomendaciones estratégicas

## Principales Hallazgos

- La evasión se concentra principalmente en los primeros meses de contrato.

- Los clientes con cargos mensuales más altos presentan mayor probabilidad de cancelación.

- El método de pago “cheque electrónico” tiene la mayor tasa de churn.

- El servicio de fibra óptica muestra niveles elevados de evasión.

- El género no representa un factor determinante en la cancelación.

## Perfil del Cliente con Mayor Riesgo de Churn

Cliente con:

- Baja antigüedad.

- Cargo mensual alto.

- Servicio de fibra óptica.

- Método de pago no automatizado.

## Recomendaciones Estratégicas

1. Implementar programas de fidelización en los primeros meses.

2. Incentivar el uso de métodos de pago automáticos.

3. Revisar la estrategia de precios y calidad en el servicio de fibra óptica.

4. Desarrollar modelos predictivos para identificar clientes en riesgo.

## Impacto del Proyecto

Este análisis permite transformar datos en decisiones estratégicas, proporcionando información clave para reducir la evasión y mejorar la rentabilidad del negocio.
